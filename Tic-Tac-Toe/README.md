# Tic Tac Toe
2 player game.<br>

By numbers:-<br>
1st box is 1,<br> 
2nd box is 2, etc.<br>
Press the required box number to put X/O.<br>

By Arrows:- (The right side keypad without number lock)<br>
1st box is 1st row 1st key,<br>
2nd box is 1st row 2nd key,<br>
3rd box is 1st row 3rd key, etc.<br>
